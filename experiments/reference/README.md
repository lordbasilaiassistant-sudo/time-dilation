# Reference builds — for labs, universities, or hobbyists with budget

These are real, well-documented experiments that measure time dilation more
precisely than the cheap two upstairs. They are kept here for completeness so
that someone with a $5,000+ budget, soldering skills, and possibly a regulatory
license can find them. **They are not the recommended starting point.**

If you're one normal person trying to engage with time dilation, go back to
[`../`](../) — the personal tracker (free) and the cloud chamber (~$20) are the
practical options.

| Experiment | Realistic minimum cost | Requires |
|---|---|---|
| [Muon detector (CosmicWatch v3X)](muon-detector/) | ~$100 (parts) + ~$50 (tools) | SMD soldering, scope (or borrow one), assembly time |
| [GPS time-dilation decoder](gps-time-dilation/) | ~$60 + a Linux box | Software-radio fluency, gnss-sdr setup |
| [Mini Hafele-Keating](hafele-keating-mini/) | $300–3,000+ | Used precision time hardware, days of driving runs |
| [Pound-Rebka style](pound-rebka/) | $10,000+ | Radioactive material license, Mössbauer drive, 22 m tower |

Each subfolder has the cited BOM and physics. They're documentation, not
expectations.
